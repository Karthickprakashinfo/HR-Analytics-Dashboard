**HR Analytics Dashboard (Excel, PostgreSQL and Power BI)**
**Overview**
This project focuses on analyzing recruitment performance using Power BI.
The dashboard provides insights into candidate flow, selection rates, and hiring efficiency.

**Key Metrics**
Total Candidates: 168
Selected Candidates: 43
Selection Rate: 25.6%
Avg Time to Shortlist: 3.36 days

**Key Insights**
Only 25.6% of candidates are selected, indicating high drop-off
Largest drop occurs at the shortlist stage
Interview-to-selection conversion is relatively strong
Shortlisting process is efficient with low average time

**Tools Used**
Power BI
DAX
Data Visualization

**Features**
Recruitment funnel analysis
Conversion rate tracking
Stage-wise time analysis
Interactive filters
